int unlink(const char *file)

Method Description

Delete or unlink (depending on destination OS) the file given in argument.
Note: The SMB server must have permission to delete the file.

file is the URL of the file to delete, in ASCIIZ.

This function returns 0 on success and -1 on error with error() returning the corresponding error code.
